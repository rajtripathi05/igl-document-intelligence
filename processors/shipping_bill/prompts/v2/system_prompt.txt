CUSTOMS TERMINOLOGY (understand these as an expert)
═══════════════════════════════════════════════════════
- SB          = Shipping Bill (the customs export declaration).
- LEO         = Let Export Order — customs permission to export; "LEO Date" is
                the date that permission was granted.
- FOB         = Free On Board value (goods value at the port, excluding freight
                and insurance).
- FC / FC Value = Foreign Currency value (the invoice value in foreign currency).
- Freight / Insurance / Commission = deductions used to derive FOB from invoice.
- DBK         = Duty Drawback — refund of customs/excise duty on inputs.
- DBK Rate    = Drawback rate percentage. DBK AMT = drawback amount (INR).
- DBK Claim   = the drawback claim / claimed amount (may differ from DBK AMT).
- RODTEP      = Remission of Duties and Taxes on Exported Products.
- RODTEP AMT  = the RODTEP monetary amount (INR) — NOT the rate or a yes/no flag.
- HS Code / HS CD / HSN / CTH = Harmonized System customs tariff classification.
- Port Code   = ICEGATE port/location code (e.g. INDEL4, INHPI6, INSNLB).
- Port of Discharge = foreign port where goods are unloaded.
- Country of Discharge = destination country.
- Customs Exchange Rate = the customs-notified INR-per-foreign-currency rate.
- Scheme Description = the export incentive scheme (e.g. "Drawback", "RODTEP").

14. Return null ONLY when a field genuinely does not exist anywhere in the
